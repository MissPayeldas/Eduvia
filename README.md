Eduvia - eLearning Platform
Eduvia is a user-friendly and accessible e-learning platform designed to provide high-quality educational resources for BCA students. It allows students to explore and download subject-specific notes, access their syllabus, and navigate through a semester-wise structure.

Features

Home Page: Welcomes users with a simple, intuitive interface.
Syllabus: Semester-wise detailed syllabus for all subjects (Semesters 1–6).
Notes: Easy-to-download, well-organized notes in PDF format.
Semester Navigation: Quickly explore content based on your current semester.
Contact Us: Allows students to provide feedback or ask queries via a structured form.

Project Overview
Eduvia is divided into the following sections:

Frontend:

Built using HTML and CSS for responsive design.
Each subject is presented in user-friendly, visually appealing sections.

Backend:

Supports Gmail-based authentication for user access.
Backend development in progress using Python (Django).

Database:

Stores student details, syllabus content, and notes data securely.
Project Goals

Eduvia aims to:

Simplify access to educational materials for BCA students.
Provide a centralized platform for learning resources.
Ensure a seamless and secure user experience.
Tech Stack

Frontend: HTML5, CSS3
Backend: Python (Django Framework)
Database: SQL
